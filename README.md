# YouTube & Twitch Volume Controller by Mouse Wheel

YouTube と Twitch の動画プレイヤー上でマウスホイールを回すことで、音量を 1% 刻みで細かく調整できるようにするブラウザ拡張機能です。

## 主な機能

* **1%単位での細かな音量調整**
  * 動画プレイヤー上にマウスカーソルを置いてホイールを回転させるだけで、音量を1%ずつ増減できます。
  * ホイール上回転（奥へ回す）：音量 +1%
  * ホイール下回転（手前へ回す）：音量 -1%
* **ページスクロールの防止**
  * 動画プレイヤー上でホイールを動かしている間、ブラウザのページ全体がスクロールするのを防ぎます。
* **自動ミュート解除**
  * ミュート（消音）状態のときにホイールを動かすと、自動的に消音が解除され音量が調整されます。
* **プレイヤーUIとの完全な同期**
  * **YouTube**: 公式の音量インジケータ（画面中央のパーセンテージ表示）や音量バーと完全連動します。
  * **Twitch**: Reactの内部状態（State）と同期し、プレイヤー右下の音量スライダーおよびアイコンの表示が自動的に追従して動きます。

## インストール方法

本拡張機能は、パッケージ化されていない「開発者モード」の拡張機能としてブラウザに読み込ませて使用します。

1. 本リポジトリのソースコードをPCにダウンロード（またはクローン）します。
2. 対象のブラウザを起動し、拡張機能の管理画面を開きます。
   * **Vivaldi**: アドレスバーに `vivaldi://extensions` と入力
   * **Google Chrome**: アドレスバーに `chrome://extensions` と入力
   * **Microsoft Edge**: アドレスバーに `edge://extensions` と入力
3. 画面右上にある **「デベロッパー モード」** のトグルスイッチを **オン** にします。
4. 画面左上に表示される **「パッケージ化されていない拡張機能を読み込む」**（または「展開された拡張機能を読み込む」）ボタンをクリックします。
5. ダウンロード（展開）したこのリポジトリのルートフォルダ（`manifest.json` が入っているフォルダ）を選択して読み込ませます。
6. すでに開いている YouTube や Twitch のタブがある場合は、一度 **ページを再読み込み（F5）** してから動画上でホイールをお試しください。

## 技術的な実装詳細

* **Manifest V3 準拠**: 最新のChrome拡張機能規格に適合しています。
* **`world: "MAIN"` での実行**: YouTubeおよびTwitchのWebページと同じJavaScript実行空間（MAINコンテキスト）で動作するため、YouTubeのプレイヤーAPIやTwitchのReact状態を書き換えるためのDOMプロトタイプ操作が可能です。
* **キャプチャフェーズのイベントリッスン**: プレイヤー上の様々なオーバーレイ要素によるイベント伝播の遮断（`stopPropagation`）を避けるため、キャプチャフェーズ（`capture: true`）でホイールイベントをインターセプトしています。
